## HOW TO USE RELATIONS

### When Creating New Content
1. Identify which Master Book chapter it serves
2. Check existing documents that feed that chapter
3. Ensure consistency across documents
4. Use existing materials as foundation

### When Updating Existing Documents
1. Check which Master Book chapter is affected
2. Ensure Master Book alignment
3. Update Master Book if needed (rare)
4. Maintain consistency across layer

### When Introducing New AI System
1. Provide Master Book (especially Chapter 7)
2. Provide relevant existing documents
3. Ensure AI understands hierarchy:
   - Master Book = Constitutional (must follow)
   - Existing docs = Operational (should follow)
   - AI outputs = Tactical (subject to both)
