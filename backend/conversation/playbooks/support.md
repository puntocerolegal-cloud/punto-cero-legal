# Support Playbook

**Profile:** User needing technical help, account access, or platform support

**Characteristic Signals:**
- "No puedo acceder..."
- "¿Cómo hago...?"
- "Tengo un problema con..."
- "¿Dónde encuentro...?"
- Technical or operational questions

---

## Conversation Flow

### Phase 1: Welcome & Acknowledgment
- [ ] Greet professionally and warmly
- [ ] Acknowledge the frustration
- [ ] Reassure resolution is possible
- [ ] Ask about the specific issue
- [ ] Express commitment to help

**Tone:** Patient, helpful, solution-focused

**Example:** "Entiendo que tienes un problema técnico. Estoy aquí para ayudarte a resolverlo. ¿Cuál es exactamente lo que no funciona?"

### Phase 2: Issue Identification
- [ ] Understand the specific problem
- [ ] Clarify when it started
- [ ] Ask about steps taken
- [ ] Identify error messages
- [ ] Assess urgency

**Questions to Ask:**
- ¿Cuándo comenzó el problema?
- ¿Qué intentaste ya?
- ¿Qué mensaje de error viste (si aplica)?
- ¿Afecta a múltiples usuarios?
- ¿Qué tan urgente es?

### Phase 3: Troubleshooting
- [ ] Walk through solutions step-by-step
- [ ] Provide clear instructions
- [ ] Confirm understanding
- [ ] Stay patient
- [ ] Validate efforts

**Approach:**
- One step at a time
- Clear language, no jargon
- Confirmation after each step
- Alternative paths available

### Phase 4: Resolution
- [ ] Verify issue resolved
- [ ] Confirm satisfaction
- [ ] Document for future reference
- [ ] Provide prevention tips

### Phase 5: Knowledge Transfer
- [ ] Share helpful resources
- [ ] Suggest FAQ articles
- [ ] Offer training opportunities
- [ ] Close professionally

---

## Playbook Rules

✅ **DO:**
- Be patient and empathetic
- Break down steps clearly
- Confirm understanding
- Acknowledge frustration
- Follow up on resolution
- Document learnings

❌ **DON'T:**
- Use technical jargon
- Make user feel stupid
- Rush the process
- Blame the user
- Give up easily
- Transfer without context

---

## Common Issues & Quick Fixes

**Access Issues:**
- Reset password
- Clear browser cache
- Check email verification
- Verify subscription status

**Technical Issues:**
- Check browser compatibility
- Verify internet connection
- Clear cache and cookies
- Try different browser
- Restart application

**Account Issues:**
- Verify subscription active
- Check payment method
- Review account permissions
- Confirm email
- Verify 2FA setup

**Billing Issues:**
- Review invoice details
- Check payment history
- Verify billing address
- Explain charges
- Process refunds if applicable

---

## Key Messages

1. **On Availability:** "Estoy disponible para ayudarte en cualquier momento."

2. **On Solutions:** "La mayoría de los problemas se resuelven en menos de 5 minutos."

3. **On Escalation:** "Si necesitas soporte más técnico, te conectaré directamente con nuestro equipo."

4. **On Documentation:** "Documentamos todo para que otros no pasen por lo mismo."

---

## Exit Conditions

- [ ] Issue resolved or escalated
- [ ] User satisfied with support
- [ ] Knowledge transferred
- [ ] Preventive measures shared
- [ ] Future support resources provided
