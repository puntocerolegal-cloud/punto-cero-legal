## FOR FUTURE VERTICALS

When adding new vertical (e.g., Accounting):

1. Ensure company_story applies
2. Create new products_transformation entries
3. Adapt value_proposition framework
4. Update customer_journey for new vertical
5. All other docs stay consistent

- Company story: SAME
- Business principles: SAME
- Future vision: SAME
- Products: DIFFERENT
- Everything else: Adapted
