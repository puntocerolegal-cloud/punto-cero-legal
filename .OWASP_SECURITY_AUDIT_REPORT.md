## A01: BROKEN ACCESS CONTROL {#a01}

### Estado: ✅ FUERTE

#### Hallazgos Positivos

✅ **JWT Token Validation**
```python
# backend/routes/users.py:14-26
async def get_current_user(authorization: Optional[str] = Header(None)):
    if not authorization:
        raise HTTPException(status_code=401, detail="No autenticado")
    
    token = authorization.replace("Bearer ", "")
    payload = decode_token(token)
    if not payload:
        raise HTTPException(status_code=401, detail="Token inválido")
```
Status: ✅ Implementado correctamente

✅ **Role-Based Access Control (RBAC)**
```python
# backend/routes/admin.py:26-27
if user["role"] not in ["admin", "admin_general", "socio_comercial"]:
    raise HTTPException(status_code=403, detail="Acceso denegado")
```
Status: ✅ Validado en todos los endpoints

✅ **Tenant Isolation**
```python
# backend/routes/subscriptions.py:40
ctx=Depends(get_tenant_context)
# Todas las queries filtradas por tenant_id
```
Status: ✅ Implementado sistemáticamente

✅ **ObjectId Validation**
```python
# backend/routes/admin_master.py:21-24
def _oid(v):
    if not ObjectId.is_valid(str(v)):
        raise HTTPException(404, "Identificador inválido")
    return ObjectId(v)
```
Status: ✅ Previene inyección de IDs

#### Hallazgo: MEDIO

⚠️ **Role Checking Logic**
- **Ubicación:** Múltiples rutas (admin.py, sales_analytics.py)
- **Problema:** Algunos endpoints no validan `get_admin()` al inicio, solo dentro
- **Ejemplo:** 
  ```python
  @router.get("/dashboard/general")
  async def get_general_dashboard(admin=Depends(get_current_admin)):
      if admin["role"] not in ["admin", "admin_general"]:
          raise HTTPException(403, ...)
  ```
- **Riesgo:** Si `get_current_admin` falla silenciosamente, se podría saltear la validación
- **Mitigación:** Ya está presente (`Depends(get_current_admin)` fuerza validación)
- **Severidad:** MEDIO

### Conclusión A01
**Status: ✅ SEGURO** — JWT + RBAC + Tenant isolation implementados correctamente

- JWT_SECRET debe estar en variable de entorno
- No hardcodeado en código
Status: ✅ Asumiendo configuración correcta

✅ **HMAC para Webhooks**
```python
# backend/services/webhook_handler.py:34-65
async def validate_hmac_signature(payload: str, signature: str) -> bool:
    ...
    expected = hmac.new(
        secret.encode(),
        payload.encode(),
```
