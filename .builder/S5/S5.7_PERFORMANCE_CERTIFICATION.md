## 🚀 CRITICAL PERFORMANCE ISSUES

### N+1 QUERY PATTERNS (12+ endpoints)

**Example:** `backend/routes/admin_ops.py:300-325`
```python
cases = await db.cases.find(query).limit(300).to_list(300)
for c in cases:
    if c.get("lawyer_id"):
        lawyer = await db.users.find_one(...)  # 300 queries!
```

**Impact:** 1 list query + 300 lookups = 301 DB hits for single endpoint

**Risk:** Querying 10M documents fills RAM, OOM kill
