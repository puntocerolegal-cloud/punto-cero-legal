### FINDING #1: UNVALIDATED STRING INJECTION IN TOKEN EXTRACTION
**Files:**
- `backend/routes/auth.py:25`
- `backend/routes/admin_ops.py:39`
- `backend/routes/admin.py:17`
- `backend/routes/accounting.py:25`
- `backend/routes/referrals.py:22`
**Severity:** CRITICAL
**OWASP:** A03:2021 – Injection

```python
# VULNERABLE CODE:
authorization = request.headers.get("Authorization", "")
token = authorization.replace("Bearer ", "")  # Simple string replace
payload = decode_token(token)
```

**Vulnerability:**
1. No validation that string actually starts with "Bearer "
2. If header is malformed, `replace()` could leave garbage in token variable
3. No error handling if decode_token fails

**Attack Vector:**
```
Header: Authorization: Bearer <valid_token>INJECTED_GARBAGE
Result: decode_token() called with invalid token
```

**Fix:**
```python
def extract_token_from_header(auth_header: str) -> str:
    if not auth_header:
        raise HTTPException(401, "Missing authorization header")
    
    parts = auth_header.split()
    if len(parts) != 2 or parts[0].lower() != "bearer":
        raise HTTPException(401, "Invalid authorization header format")
    
    return parts[1]
```

### FINDING #3: AGGREGATION PIPELINE INJECTION
**File:** `backend/routes/admin_ops.py:692-716`
**Severity:** CRITICAL
**OWASP:** A03:2021 – Injection

```python
# VULNERABLE PATTERN:
pipeline = [
    {"$match": {"role": "lawyer"}},
    {"$group": {"_id": "$country", "count": {"$sum": 1}}},  # country from DB field
]
countries_cur = db.users.aggregate(pipeline)
```

**Vulnerability:**
While this specific code is safe (hardcoded pipeline), if ANY pipeline stage uses user input:

```python
# UNSAFE EXAMPLE (if this existed):
pipeline = [
    {"$match": field_from_user_input},  # INJECTION POINT
    {"$group": {"_id": group_field, "count": {"$sum": 1}}}
]
```

**Finding:**
No user-controlled aggregation parameters found, but pattern is high-risk.

**Recommendation:**
```python
# Whitelist allowed grouping fields
ALLOWED_GROUP_FIELDS = {"country", "role", "status", "plan_id"}

@router.post("/stats")
async def get_stats(group_by: str = Query(...)):
    if group_by not in ALLOWED_GROUP_FIELDS:
        raise HTTPException(400, f"Invalid group field: {group_by}")
    
    pipeline = [
        {"$group": {"_id": f"${group_by}", "count": {"$sum": 1}}}
    ]
    return await db.users.aggregate(pipeline).to_list(None)
```

### FINDING #4: INSUFFICIENT AUTHORIZATION ON FORM ENDPOINT
**File:** `backend/routes/cases.py:434-454`
**Severity:** CRITICAL
**OWASP:** A01:2021 – Broken Access Control

```python
@router.get("/form/{token}")
async def get_client_form(token: str, db: AsyncIOMotorDatabase = Depends(get_db)):
    """Datos básicos del caso para precargar el formulario público (sin auth)."""
    case = await db.cases.find_one({"client_form_token": token})
    # Returns case details with NO authentication
    return {
        "case_number": case.get("case_number"),
        "title": case.get("title"),
        "client_name": case.get("client_name"),
    }

@router.post("/form/{token}")
async def submit_client_form(token: str, payload: dict, db: AsyncIOMotorDatabase = Depends(get_db)):
    """El cliente envía sus datos - NO AUTH REQUIRED"""
    case = await db.cases.find_one({"client_form_token": token})
    # No validation that token is valid UUID format
    # No rate limiting
    # No expiration check on token
```

**Vulnerabilities:**
1. **Token format not validated** — UUID format not checked
2. **No rate limiting** — Attacker can brute-force tokens
3. **No expiration check** — Old tokens never expire
4. **No scope validation** — Any token grants access to ANY case data

**Attack Scenario:**
```
1. Attacker generates 1000 random UUIDs
2. Sends GET /cases/form/{uuid} for each
3. Discovers valid case tokens by observing response structure
4. Extracts case numbers, client names, materia
5. Submits fake client data with malicious content
```

**Fix:**
```python
import secrets
from datetime import datetime, timedelta

@router.get("/form/{token}")
async def get_client_form(token: str, db: AsyncIOMotorDatabase = Depends(get_db)):
    # Validate token format (UUID hex, 32 chars)
    if not (isinstance(token, str) and len(token) == 32 and all(c in '0123456789abcdef' for c in token)):
        raise HTTPException(400, "Invalid token format")
    
    case = await db.cases.find_one({
        "client_form_token": token,
        "client_form_token_expires_at": {"$gt": datetime.utcnow()}  # Check expiration
    })
    if not case:
        raise HTTPException(404, "Form not found or expired")
    
    return case_form_data
```

**Files:**
- `backend/routes/admin_ops.py:704` — $unwind without bounds
- `backend/routes/billing_admin.py:137-155` — Aggregation without input validation
**Severity:** HIGH
**OWASP:** A03:2021 – Injection

```python
case_pipeline = [

```

### FINDING #21: MISSING INPUT VALIDATION ON DOCUMENT CONTENT
**File:** `backend/routes/cases.py:471-479`
**Severity:** MEDIUM
**OWASP:** A04:2021 – Insecure Design

```python
for doc in payload.get("documents", []):
    await db.documents.insert_one({
        "name": doc.get("name", "documento_cliente"),
        "mime": doc.get("mime"),  # User can set mime type
        "content_b64": doc.get("content_b64"),  # Base64 could be anything
        "size_bytes": doc.get("size_bytes", 0),  # Could be fraudulent
    })
```

**Vulnerabilities:**
1. No validation of file size (user could claim 1GB document)
2. No whitelist of allowed MIME types
3. No validation of base64 encoding
4. No antivirus/malware scan

**Fix:**
```python
import base64

ALLOWED_MIME_TYPES = {
    "application/pdf",
    "image/jpeg",
    "image/png",
    "application/msword",
    "application/vnd.openxmlformats-officedocument.wordprocessingml.document"
}

async def submit_client_form(token: str, payload: dict, ...):
    for doc in payload.get("documents", []):
        # Validate MIME type
        if doc.get("mime") not in ALLOWED_MIME_TYPES:
            raise HTTPException(400, f"File type not allowed: {doc.get('mime')}")
        
        # Validate file size (max 10MB)
        if doc.get("size_bytes", 0) > 10 * 1024 * 1024:
            raise HTTPException(400, "File too large")
        
        # Validate base64
        try:
            content = base64.b64decode(doc.get("content_b64", ""), validate=True)
        except Exception:
            raise HTTPException(400, "Invalid file encoding")
```
